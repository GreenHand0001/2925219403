# 项目简介

## 核心功能
- 基于YOLOv8的集装箱缺陷检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 集装箱运输行业的安全检测
- 港口和物流中心的集装箱质量检查
- 自动化缺陷识别与分类
- 批量集装箱图像分析
- 实时视频流监控

## 技术特点
- 使用YOLOv8目标检测算法
- 包含7731张集装箱缺陷数据集
- 支持Python 3.9环境
- 集成opencv-python、PyQt5、torch1.9等库
- 提供图形用户界面
- 包含完整源码、标注数据集、训练好的模型及运行教程
- 支持跨平台部署（Pycharm+Anaconda或VSCode+Anaconda）
